# Weather Poster

This project lets you create A4-size poster of the weatherforecast for a specific position

## Get Started

1. clone repo
2. install dependencies `pnpm install`
3. generate post, example: `npx tsx src/generate.ts --lat 59.93 --lon 10.72 --altitude 90 --location "Oslo" --days 4`
